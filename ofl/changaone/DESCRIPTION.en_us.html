Changa One is intended for titles, with its short ascenders and descenders and a set of lowercase letters inscribed within a square. The uppercases case gains slightly more in height and develops its morphology in a single height in order to make it possible to create text composition with minimum line spacing.

Its counter-shapes are rectangular, featuring small curvatures in opposite vertexes which accompany and break the shapes, thus evoking a modern style.

A script type by Eduardo Tunni, and more documentation can be found at www.tipo.net.ar
