FONTLOG for the Inconsesi fonts

This file provides detailed information on the Inconsesi
Font Software.

This information should be distributed along with the 
Inconsesi fonts and any derivative works.

Basic Font Information

Alexei Vanyashin, the primary designer of the family, wrote:

Inconsesi is a monospaced design based on 
Inconsolata by Raph Levien. It supports the WGL-4 encoding.

Inconsesi is a Unicode typeface family that supports 
languages that use the Latin script and its variants, 
and could be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: WGL-4
