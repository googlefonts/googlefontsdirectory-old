FONTLOG for the Keania One font

This file provides detailed information on the Keania One font
Software.

This information should be distributed along with the Keania One and any derivative works.

Basic Font Information

Keania One is a Unicode typeface family that supports languages 
that use the Latin script and its variants, and could be expanded 
to support other scripts.

There are three source files:

1. KeaniaOne-Regular.vfb Drawing contours

2. KeaniaOne-Regular-OTF.vfb Merged contours and 
   optimized file corresponding to the OTF file

3. KeaniaOne-Regular-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file

Source files are available from
http://code.google.com/p/googlefontdirectory/
