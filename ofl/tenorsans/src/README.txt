Tenor Sans - Humanist sans serif typeface, designed by Denis Masharov.
Designed to set the main text, but can also be applied in the headlines,
optimized for print and web, has excellent legibility characteristics in its letterforms.
Contains an extended Latin and extended Cyrillic alphabets,  supports these codepages:
MacOS Roman
MacOS Central Europe
MacOS Croatian
MacOS Cyrillic
MacOS Icelandic
MacOS Romanian
MacOS Turkish

MS Windows 1252 Western (ANSI)
MS Windows 1250 Central European
MS Windows 1251 Cyrillic
MS Windows 1254 Turkish
MS Windows 1257 Baltic

ISO 8859- 1 Latin 1 (Western)
ISO 8859- 2 Latin 2 (Central Europe)
ISO 8859- 3 Latin 3 (Tu, Malt, Gal, Esp)
ISO 8859- 4 Latin 4 (Baltic)
ISO 8859- 5 Cyrillic
ISO 8859- 9 Latin 5 (Turkish)
ISO 8859-13 Latin 7 (Baltic 2)
ISO 8859-14 Latin 8 (Celtic)
ISO 8859-15 Latin 9
ISO 8859-16 Latin 10
