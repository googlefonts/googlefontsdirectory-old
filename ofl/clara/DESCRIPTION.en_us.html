Proyecto Demo is a collaboration between type designers in Chile and Argentina. The first font released by the project is Clara. Designed by Alejandro Paul, Alejandro Lo Celso, Eduardo Manso, Eduardo Tunni, Jose Scaglione, Pablo Cosgaya, Francisco Galvez, Rodrigo Ramirez, Tono Rojas, Kote Soto, Luciano Vergara and Felipe Caceres. Coordinated by Pancho Galvez, Cristian Gonzalez Saiz, Daniel Berczeller and Andreu Balius.

Be sure to see the project homepage, www.proyectodemo.cl
