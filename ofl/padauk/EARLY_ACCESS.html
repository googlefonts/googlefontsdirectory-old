Padauk (Myanmar)

Padauk is a fully capable Unicode 6 font supporting all the Myanmar characters in the standard. Thus it provides support for minority languages as well, in both local and Burmese rendering style.

Padauk includes the necessary Graphite smarts to render Myanmar script correctly. Thus if you want to use this font you will need the Graphite libraries and Graphite capable applications or graphics extensions. For Web browsers this includes the latest versions of Firefox

Padauk also has OpenType tables which allow it to be rendered using a basic shaping engine and so allows the font to be used in such applications as Word and browsers that support OpenType shaping.
