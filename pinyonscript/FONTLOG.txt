FONTLOG for the Pinyon Script font

This file provides detailed information on Pinyon Script 
Font Software.This information should be distributed along 
with the Pinyon Script font and any derivative works.

Basic Font Information

Pinyon Script is a Unicode typeface family that supports 
languages that use the Latin script and its variants, and 
could be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1

To contribute to the project contact Eben Sorkin at 
sorkineben@gmail.com
