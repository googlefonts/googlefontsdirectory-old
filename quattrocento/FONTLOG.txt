FONTLOG for the Quattrocento Roman fonts

This file provides detailed information on the Quattrocento Roman Font Software.
This information should be distributed along with the Quattrocento Roman fonts
and any derivative works.

Basic Font Information

Quattrocento Roman is a Unicode typeface family that supports languages that
use the Latin script and its variants, and could be expanded to support other
scripts.

More specifically, this release supports the following Unicode ranges: Latin-1

Documentation can be found at www.impallari.com/Quattrocento Roman
To contribute to the project contact Pablo Impallari at impallari@gmail.com
