FONTLOG for the Clara fonts

This file provides detailed information on the Clara Font Software.
This information should be distributed along with the Clara fonts
and any derivative works.

Basic Font Information

Proyecto Demo is a collaboration between type designers in 
Chile and Argentina. 

The first font released by the project is Clara. 

Designed by Alejandro Paul, Alejandro Lo Celso, Eduardo 
Manso, Eduardo Tunni, Jose Scaglione, Pablo Cosgaya, 
Francisco Galvez, Rodrigo Ramirez, Tono Rojas, Kote 
Soto, Luciano Vergara and Felipe Caceres. 
Coordinated by Cristian Gonzalez Saiz, Daniel Berczeller 
and Andreu Balius.

Be sure to see the project homepage, www.proyectodemo.cl

Clara is a typeface family that supports Unicode language 
range: Latin-1

To contribute to the project contact Francisco Fuentes 
(franciscofuentes@udd.cl)
