FONTLOG for the Montaga font.

This file provides detailed information on the Montaga 
font Software.

This information should be distributed along with the 
Montaga fonts and any derivative works.

Basic Font Information

Montaga is a Roman font.

Montaga is evolving and will be updated.
As of now there is only Regular style. 

Montaga is a work in progress and
will be improved regularly. This means you can request
improvements and even fund specific features 
if they are outside of the current scope of work.

For more information follow: 

http://twitter.com/ale_guez

More specifically, this release supports the following Unicode
ranges: Latin-1

To contribute to the project contact Alejandra Rodriguez at 
ale_guez@yahoo.com.ar

ChangeLog

1 January 2012 (Alejandra Rodriguez) Montaga v 1.1

- Altered and improved 90% of the glyphs. Some in minor ways 
  and other in fairly significant ways. For example: 
  The lc a was a small change.
  The changes to the numbers corrected weight consistency issues 
  were more significant. 
- Added glyphs to support texts originating from Windows
  ANSI and Mac Roman encodings. 
- Adjusted diacritics further
